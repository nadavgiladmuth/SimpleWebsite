About This Site

Why I Made This Website

I created this site to showcase my naturalist photography and to put together a clean, functional webpage from scratch. I wanted to make sure my photos were properly credited, included alternative text for accessibility, and had a layout that worked well across different screen sizes. Beyond just displaying my work, this site was also a way for me to practice web development—figuring out how to make everything look right while keeping the code organized and readable.

Building the Site

I built this site using HTML for structure and CSS for styling. I used Visual Studio Code as my text editor, which was really helpful with features like syntax highlighting. For debugging, I used the browser’s developer tools to adjust things on the fly. The layout is simple, but it includes everything needed to keep the site functional and easy to navigate. I made sure to add alt text to all the images and ensured the text contrasted well with the background for better accessibility.

For design, I kept it clean and responsive. I used flexbox to align elements properly and media queries to make sure the site looks good on both phones and computers. The biggest challenge was making sure everything looked right on different screen sizes, but testing and tweaking along the way helped a lot.
