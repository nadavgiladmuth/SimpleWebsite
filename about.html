About This Site

Why I Made This Website

I created this site to showcase my naturalist photography and make it easier to remember and document my observations. I wanted to give each photo a place to be properly credited, with alternative text for accessibility, and ensure the site had a clean, fun vibe. I also wanted to provide useful information for users by showing where species are from, so it’s not only a visual showcase but also a helpful reference. It’s also a way for me to keep track of when and where I took each photo.

Building the Site

To build the site, I started with HTML to create the basic structure, adding sections like headers, images, and links. After that, I used CSS to style the site and give it a clean, fun look. I made sure all images had alt text for accessibility and tested the site on both mobile devices and desktop screens. I focused on making the text easy to read and spacing everything out nicely. For the process of adding information about the observations, I had a split-screen setup open where I would copy and paste the information from my iNaturalist posts, which included all the relevant details about the dates, locations, common and scientific names. Overall, it was a process of trial and error, but I learned a lot about organizing and styling websites along the way!
